Optimización

Puede ser definido como el proceso de obtener las condiciones que producen el máximo o mínimo valor de una función.

La optimización en ingeniería se debe empezar desde el diseño, es mejor prevenir errores que corregirlos, como ingenieros siempre es necesario aspirar a que el sistema diseñado sea el mejor esto conlleva que se cumpla con eficiencia, versatilidad e invariablemente bajo costo. Para lograr una optimización se tiene que plantear un problema mismo que puede ser de dos tipos con restricciones o sin restricciones a partir de estas dos se generan una gran variedad de clasificaciones, dependiendo del tipo de este se puede hacer uso de métodos o técnicas para partir de un estado inicial a un estado inicial que satisfaga el mínimo o máximo de una función según sea su caso.
